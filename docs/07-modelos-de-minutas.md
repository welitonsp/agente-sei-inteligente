# Modelos de minutas

## Regra geral

Toda minuta gerada pelo agente e rascunho. Ela deve ser revisada, ajustada e aprovada por humano antes de qualquer uso oficial.

## Campos comuns

```text
Processo SEI:
Documento de origem:
Interessado:
Assunto:
Resumo:
Providencia sugerida:
Responsavel:
Prazo:
Fundamento/contexto:
Texto da minuta:
Observacoes para revisao:
```

## Modelo 1 - Despacho de encaminhamento

```text
DESPACHO

Considerando o teor do documento/processo em referencia, que trata de [assunto], encaminhe-se a presente demanda a [unidade/responsavel] para conhecimento e adocao das providencias cabiveis.

Solicita-se especial atencao ao prazo de [prazo], quando aplicavel.

Apos, retornem os autos para acompanhamento/arquivo, conforme o caso.
```

## Modelo 2 - Oficio de resposta

```text
OFICIO

Assunto: [assunto]

Senhor(a) [destinatario],

Em atencao ao expediente mencionado, informo que [resumo da resposta/providencia].

Permanecemos a disposicao para eventuais esclarecimentos.

Respeitosamente,
```

## Modelo 3 - Comunicacao interna

```text
COMUNICACAO INTERNA

Assunto: [assunto]

Encaminha-se para conhecimento e providencias o teor da demanda referente a [resumo].

Providencia esperada: [providencia].
Prazo: [prazo].
Responsavel: [responsavel].
```

## Modelo 4 - Convocacao ou ciencia de evento

```text
COMUNICADO

Assunto: [nome do evento]

Informo a realizacao de [evento], conforme dados abaixo:

Data:
Horario:
Local:
Publico-alvo:
Providencia:

Solicita-se ampla ciencia aos interessados e observancia das orientacoes constantes do documento de origem.
```

## Modelo 5 - Registro de providencia

```text
REGISTRO DE PROVIDENCIA

Demanda analisada:
Origem:
Assunto:
Providencia adotada:
Pendencia remanescente:
Responsavel:
Prazo:
Observacao:
```

## Validacoes obrigatorias

Antes de apresentar uma minuta como pronta para revisao, o agente deve checar:

1. Se o tipo de minuta foi escolhido.
2. Se assunto e origem estao claros.
3. Se prazo foi preservado corretamente.
4. Se nao inventou fundamento legal, autoridade ou decisao.
5. Se nomes, datas e unidades foram mantidos como no documento de origem.
6. Se ha alerta de baixa confianca quando faltar informacao.

## Proibicoes para o minutador

O minutador nao deve:

1. Criar decisao final sem base no documento.
2. Assumir responsabilidade por autoridade humana.
3. Declarar cumprimento de providencia nao confirmada.
4. Alterar prazo.
5. Omitir informacao sensivel para tomada de decisao.
6. Usar linguagem informal em documento administrativo.
