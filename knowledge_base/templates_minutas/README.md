# Templates locais de minutas

Estes modelos sao usados como referencia do minutador zero custo.

Regras:

1. Todo texto gerado e rascunho.
2. Revisao humana e obrigatoria.
3. O Agente 19 nao assina, nao tramita, nao envia e nao conclui processo.
4. Campos desconhecidos devem permanecer como `[PREENCHER ...]`.
5. Fundamento legal, autoridade, unidade e prazo nao devem ser inventados.
